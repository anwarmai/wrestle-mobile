Selection screens

- `GenderSelect.tsx` — simple gender picker component.
- `CountrySelect.tsx` — country picker (placeholder list).
- `WeightSelect.tsx` — weight class picker.

Integration:
- These components are lightweight and can be wired with your preferred navigation library (React Navigation) or simple stateful flows in `App.tsx`.
- They accept an optional `onSelect` callback to receive the selected value.
